# GardezinOS-GSI
GardezinOS is a Lineage-Based GSI, It is made for all Treble-Compliant, A/B Android Devices, for more info, readme


# README

GardezinOS is a LineageOS Based ROM, It is made extremely lightweight while still being usable
it has two variants, FOSS and GAPPS, Please Note that all Google apps are burnt into the ROM.
The FOSS edition is the baseline and might receive more updates then the other GAPPS, 
I recommend you to use the FOSS one if you want to be free from Google, however if you are 
going to Daily Drive this OS, It is recommended to use the GApps edition

# INCLUDED FOSS APPS

* F-Droid (F-Droid has many FOSS apps that can replace your google services)
* Aurora Store (A Client for Play Store, can get most apps that dont rely directly on google services, update by F-Droid)
* Fossify Gallery (A Lightweight Gallery application)
* Open Cam (This is the app that will make the camera work because it supports many HALs)

# INCLUDED GAPPS

* Play Store (Just your usual Play Store)
* Google Camera (Just your usual Camera)
* Gallery GO (A Lightweight, Google-Made app)

# THE ROM IS EXT4 

The Pros of EXT4:
* Full RW support with root
* a stable linux filesystem
* Devices can be formatted as EXT4 (SD Cards, External USBs)

The Cons of EXT4
* The system image is a little large unlike EROFS Formatting
* Unsupported by a few older TWRP versions (which makes it not work after the rom is flashed)
